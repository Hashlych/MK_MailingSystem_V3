Import

Import contacts from a file.
This feature is accessible via the menu entry Contacts > Import.

Supported file types: CSV (.csv)

CSV file format

First line represents columns headers.

"First name","Last name","Email","Group"

Example: "Master","Chief","master.chief@halo.net","Elite"
Note: Group is optional.
If no group is provided or if the group doesn't exists, contact will be assigned to the default group.

Export

Export selected contacts and groups of contacts to one of the supported format.
This feature is accessible via the menu entry Contacts > Export.

Supported file type: CSV (.csv), Text (.txt), HTML (.html)
